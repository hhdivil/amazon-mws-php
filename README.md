amazon-mws-php
===========================

### Amazon Marketplace Web Service libraries

Installation
------------

Install [Composer](http://getcomposer.org/) and add amazon-mws to your `composer.json`:

    "require": {
      "hhdivil/amazon-mws-php":"dev-master"
    }
